## 🚀 Featured Projects

### [playwright-ui-automation-framework](https://github.com/GonzaloBaldiviezo/playwright-ui-automation-framework)
> Comprehensive E2E UI automation framework built with Playwright and TypeScript.

- **Page Object Model (POM)** architecture for maintainable, scalable tests
- E2E test coverage: login, cart, checkout, product sorting, logout
- **Visual regression testing** with screenshot comparisons
- **Cross-browser testing** across Chromium, Firefox, and WebKit
- CI/CD pipeline with **GitHub Actions** (runs on push and pull request)
- HTML reports with screenshots, videos, and traces on failure
- Test tagging for selective execution (`@e2e`, `@visual-regression`, `@failing-demo`)

---

### [playwright-api-automation-framework](https://github.com/GonzaloBaldiviezo/playwright-api-automation-framework)
> Production-ready API test automation framework built on Playwright's native request capabilities.

- Reusable **ApiClient abstraction** covering all HTTP methods
- **Shared Playwright fixtures** with clean fixture injection pattern
- Fail-fast environment validation to catch config issues before tests run
- Layered test strategy: smoke, positive, negative, and failing-demo suites
- Tag-driven execution (`@smoke`, `@positive`, `@negative`, `@failing-demo`)
- CI/CD pipeline with **GitHub Actions**
- HTML reports on every run
